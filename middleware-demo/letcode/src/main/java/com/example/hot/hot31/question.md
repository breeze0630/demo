
###  31. 下一个排列

##### 题干

实现获取 下一个排列 的函数，算法需要将给定数字序列重新排列成字典序中下一个更大的排列（即，组合出下一个更大的整数）。

如果不存在下一个更大的排列，则将数字重新排列成最小的排列（即升序排列）。

必须 原地 修改，只允许使用额外常数空间。


`示例 1`

```text
输入：nums = [1,2,3]
输出：[1,3,2]
```


`示例 2`

```text
输入：nums = [3,2,1]
输出：[1,2,3]
```


`示例 3`

```text
输入：nums = [1,1,5]
输出：[1,5,1]
```


`示例 4`

```text
输入：nums = [1]
输出：[1]
```
