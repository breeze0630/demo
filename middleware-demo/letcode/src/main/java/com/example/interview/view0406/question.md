### 面试题 04.06. 后继者

```text
设计一个算法，找出二叉搜索树中指定节点的“下一个”节点（也即中序后继）。

如果指定节点没有对应的“下一个”节点，则返回null。

```

##### 示例 1:

```text
输入: root = [2,1,3], p = 1

  2
 / \
1   3

输出: 2
```


##### 示例 2:

```text
输入: root = [5,3,6,2,4,null,null,1], p = 6

      5
     / \
    3   6
   / \
  2   4
 /   
1

输出: null
```


```text

中序后继的下一个节点：
    中序遍历，先找左子树，再找父节点，再找右子树
    
      5
     / \
    3   6
   / \
  2   4
 /   
1

这棵树的遍历循序： 1 -> 2 -> 3 -> 4 -> 5 -> 6

      8
     / \
    6   9
   / \   \
  2   7   10
 / \ 
1   3

这棵树的遍历顺序： 1 -> 2 -> 3 -> 6 -> 7 -> 8 -> 9 

二叉搜索树中特定节点的下一个节点是就是指中序后继，这里有2种大的情况
1. 指定节点没有右子树，下一个节点只能在其父节点中
2. 指定节点有右子树，则寻找右子树的最左子节点，因为最左是右子树中最小的
    找右子树的最左子节点时，如果向下找到了右子树，则保留父节点(因为是递归深度遍历，所以向右遍历了，说明此子节点一定没有左节点，
    所以记录父节点为parent，而如果是找到了子节点，则不需要记录)，这里在右节点为空的时候就返回结果，根据前面是否有记录的节点，
    即 parent!=null 时返回 parent，parent == null 时 返回当前节点
    最后对比找到的后继节点是否是指定节点，如果是则返回指定节点的右子树（当指定节点只有一个右孩子时，返回的其实是指定节点），
    否则正常返回找到的节点


```
